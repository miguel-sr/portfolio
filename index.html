Sobre mim

Olá visitante! Meu nome é Miguel! Meu interesse pela área de desenvolvimento para web começou quando eu ainda estava aprendendo a lógica de programação - em 2019 - e a interface onde o programa era executado não era tão bonita. Então conheci esse mundo e aprendi que por meio do Javascript seria possível escrever meus códigos e ainda usar o HTML e CSS para construir uma interface mais amigável.

Acabei dando uma pausa do mundo de desenvolvimento e somente em 2021 voltei a estudar sobre o assunto, desde então só venho aprimorando meus conhecimentos mais e mais.
